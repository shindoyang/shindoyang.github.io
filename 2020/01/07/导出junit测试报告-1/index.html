导出junit测试报告

idea导出junit测试报告

目前平台多采用intellij idea + junit 进行单元测试，下面提供生成单元测试报告及覆盖率测试报告的方法：

单元测试报告：

使用mvn Surefire + antrun插件方案

方法：仅需在对应项目的pom.xml文件加入以下插件：

在idea的终端窗口输入以下命令，回车执行：
mvn clean && mvn surefire-report:report -DskipTests=false

命令执行完后，会发现在target目录下多出：site及surefire-reports 两个文件夹
其中/target/site/surefile-report.html 和 /target/surefire-reports/indexhtml即为生成的单元测试报告。可以直接使用浏览器打开。

注意：其实两个文件夹的内容一致，区别在于界面的美观程度，从个人审美评判，建议采用/target/surefire-reports/indexhtml这份报告。

使用 jacoco 插件方案

方法：仅需在对应项目的pom.xml文件加入以下插件：

在idea的终端窗口输入以下命令，回车执行：
mvn clean package

命令执行完后，会发现在target目录下多出：jacoco文件夹
其中/target/site/jacoco/index.html 即为生成的覆盖率测试报告。可以直接使用浏览器打开。
